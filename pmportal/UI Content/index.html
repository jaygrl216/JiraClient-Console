About

PM-Portal is a project management tool initially developed by interns working for SGT Inc. The tool pulls together project information from other project management tools such as Jira or Bugzilla and displays a project's status or other metrics. The tools uses the agile scrum methodology, dividing various work phases into sprints which seek to satisfy user story requirements. As of 7/25/16, Jira is the only tool supported.
