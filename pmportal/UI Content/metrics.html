What does this mean?

Schedule Estimation Accuracy (SEA) is a ratio of the time
it took to complete a task divided by the time it was
estimated to take. This is calculated sprint-by-sprint.

Effort Estimation Accuracy (EEA) is a ratio of the effort
it took to complete a task divided by the effort estimated
initially. This is calculated sprint-by-sprint using a metric
called "Story Points". If a sprint contains issues
without any story points, the EEA will always default to 1.

Bug count is the number of issues listed as a "Bug" in a sprint.

The prediction model uses the Theil-Sen estimator to create
a simple linear regression model which tends to ignore
outliers, providing greater accuracy over smaller data sets.
