PMPortal Information

PM-Portal is a project management tool initially developed by interns working for SGT Inc. The tool pulls together project information from other project management tools such as Jira or Bugzilla and displays a project's status or other metrics. The tools uses the agile scrum methodology, dividing various work phases into sprints which seek to satisfy user story requirements. Currently, Jira is the only tool supported. This application has many features that will support Project Managers in managing and overseeing projects.


These features include:

Graphs
This application contains many visual graphics so you can see information about a projct in an interesting way. Some graphs include, a progress graph, an issue tracking graph and multiple metrics graphs.

Metrics
The metrics page shows detailed calculated information about individual and all projects. This includes scheduled estimated accuracy(SEA), effort estimation accuracy(EEA), bug count and more.
Backlog Information
The backlog information asset shows the user issues that have been in the backlog of their project for a long period of time. This helps PMs see what issues they need to work on soon.
